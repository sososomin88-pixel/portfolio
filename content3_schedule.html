정상 동작 CASE

CASE 1: 부하를 사용하지 않음. PV 발전 없음. ESS SOC 30% 상태. EV 충전 요구

CASE 2: 부하를 사용하지 않음. PV 발전 없음. ESS SOC 31%~90% 상태. 피크부하시간 외

CASE 3: 부하를 사용하지 않음. PV 발전. ESS SOC 31%~90% 상태. 피크부하시간

CASE 4: 부하 사용(0kW~40kW). PV 발전 없음. ESS SOC 31%~90% 상태. 피크부하시간 외

CASE 5: 부하 사용(41kW~50kW). PV 발전 없음. ESS SOC 31%~90% 상태. 피크부하시간

CASE 6: 부하 사용(0kW~40kW). PV 발전. ESS SOC 31%~90% 상태. 피크부하시간 외

CASE 7: 부하 사용(41kW~50kW). PV 발전. ESS SOC 31%~90% 상태. 피크부하시간
